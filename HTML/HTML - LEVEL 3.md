```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Page Layout Example</title>
</head>
<body>
  <header>
    <h1>Welcome to My Website</h1>
    <nav>
      <!-- Navigation links here -->
    </nav>
  </header>

  <main>
    <section>
      <h2>About Us</h2>
      <p>This is a section describing the website.</p>
    </section>

    <article>
      <h2>Latest Article</h2>
      <p>This is an article on the page.</p>
    </article>

    <aside>
      <h2>Sponsored</h2>
      <p>Ad content or links go here.</p>
    </aside>
  </main>

  <footer>
    <p>&copy; 2025 My Website</p>
  </footer>
</body>
</html>

```

**==Revisiting Anchor tag==**
```html
<a href="https://google.com"> Google </a>
<!-- opens in same tab -->

<a href="https://google.com" target="_main"> Google </a>
<!-- opens in new tab -->
```
